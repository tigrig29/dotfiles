# TigRig Dotfiles

このリポジトリは、TigRig の個人用設定ファイル（Dotfiles）を管理するものです。

PowerShell, Neovim, WezTerm, Starship などの設定が含まれています。

## PC 初期セットアップ時の手順

1. このリポジトリをダウンロード
   - `D:\TigRig\repos\ghq\github.com\tigrig29\dotfiles` に展開
   - ※ ghq で管理するため。初期セットアップ時点では git も ghq もインストールされていないため、手動でダウンロードする。
2. PowerShell を管理者権限で開き、セットアップスクリプトを実行
   ```powershell
   cd ~/.dotfiles
   .\setup-windows.ps1
   ```
